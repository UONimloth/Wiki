<h1>Yew Bow</h1>

![](https://uo-nimloth.net/tenancy/assets/wiki/posts/attachments/SdYcyhRc5GAddvq3OsoUp53ae450VQFQp5Dnc45e.png)

<p>Yew Bow, tamamen PvM odaklı özel bir uzun yay tipidir. Sadece yaratıklar ve hayvanlar üzerinde kullanılabilmekte olup, <strong style="color: #410000;">oyunculara karşı kullanılamaz</strong>.</p>
<p>Avcılık, PvM görevleri, boss mücadeleleri ve open-world farm alanlarında yüksek bir etkililik sunar.</p>

<h2>Perfect Shot Mekaniği</h2>
<p>Yew Bow kullanımında Tactics yetenek seviyenize göre aktifleşebilen Perfect Shot bonusu bulunmaktadır.</p>
<p>Bu bonus devreye girdiğinde, verdiğiniz hasar normal hasarın <strong style="color: #410000;">3 katına</strong> çıkar. Hunter Mastery 20. seviyeyi tamamladığınızda Perfect Shot hasarı <strong style="color: #410000;">4 katına</strong> çıkmaktadır.</p>
<p>Bu özellik, özellikle yüksek dirençli yaratıklar üzerinde ciddi bir tehdit oluşturur ve kritik mücadelelerde büyük avantaj sağlar.</p>

<h2>Üretim</h2>
<p>Yew Bow, <strong style="color: #410000;">Bowcrafting</strong> yeteneğine sahip karakterler tarafından üretilebilmektedir. Üretim için:</p></p> 
<p>• 50 Yew Log</p>
<p>• 20 Dragon Hides</p>
<p>• 98.1 Bowcrafting</p>
<p>• 10 Craft Mastery XP gerekmektedir.</p>
<p>Ayrıca üretim için bir Blueprint zorunludur.</p>
<p>Bu Blueprint, <strong style="color: #410000;">Craft Mastery Sistemi’nin 2. seviyesi tamamlandığında</strong> elde edilmektedir.</p>

<h2>Dayanıklılık & Eskime</h2>
<p>Yew Bow <strong style="color: #410000;">Newbified değildir</strong> ve kullanım sırasında hasar alıp eskimektedir.</p>
<p>Uzun süreli kullanımlarda dayanıklılığı azalır ve yeterince yıprandığında tamamen kırılabilir.

<script>
document.querySelectorAll(".zebra-table th").forEach((header, colIndex) => {
    let asc = true;

    header.addEventListener("click", () => {
        const table = header.closest("table");
        const tbody = table.querySelector("tbody");
        const rows = Array.from(tbody.querySelectorAll("tr"));

        // Tüm header'lardaki ikonları temizle
        document.querySelectorAll(".zebra-table th").forEach(th => {
            th.classList.remove("active");
            th.innerHTML = th.innerHTML.replace(/ ▲| ▼/g, "");
        });

        rows.sort((a, b) => {
            let valA = a.children[colIndex].innerText.trim();
            let valB = b.children[colIndex].innerText.trim();

            let numA = parseFloat(valA.replace(",", "."));
            let numB = parseFloat(valB.replace(",", "."));

            if (!isNaN(numA) && !isNaN(numB)) {
                return asc ? numA - numB : numB - numA;
            }

            return asc
                ? valA.localeCompare(valB)
                : valB.localeCompare(valA);
        });

        asc = !asc;

        // Aktif kolon işaretleme
        header.classList.add("active");
        header.innerHTML += asc ? " ▼" : " ▲";

        rows.forEach(row => tbody.appendChild(row));
    });
});
</script>
